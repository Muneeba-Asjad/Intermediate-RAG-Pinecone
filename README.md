# Intermediate RAG with Pinecone

## Project Overview
This project implements an Intermediate Retrieval-Augmented Generation (RAG) system using Pinecone vector database.

## Features
- PDF document loading
- Text chunking
- Vector embeddings
- Pinecone database integration
- Document retrieval
- AI response generation

## Project Structure

- app.py - Main application file
- pdf_loader.py - PDF loading
- chunker.py - Text splitting
- pinecone_db.py - Pinecone database handling
- retriever.py - Document retrieval
- generator.py - AI response generation
- config.py - Configuration settings
- logger.py - Logging
- requirements.txt - Required libraries

## Installation

Install dependencies:

pip install -r requirements.txt

## Technologies Used

- Python
- Pinecone
- Retrieval-Augmented Generation (RAG)
- Large Language Models (LLMs)
- Vector Database
